graphem.datasets

module documentation

Real-world dataset loader for Graphem.

This module provides functions to download and load standard large graph datasets from various sources including SNAP (Stanford Network Analysis Project), Network Repository, and other public graph repositories.

Class DatasetLoader Base class for dataset loaders.
Class NetworkRepositoryDataset Loader for datasets from the Network Repository.
Class SemanticScholarDataset Loader for Semantic Scholar citation network datasets.
Class SNAPDataset Loader for datasets from the Stanford Network Analysis Project (SNAP).
